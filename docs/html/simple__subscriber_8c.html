The client's refresher. This function triggers back-end routines to handle ingress/egress traffic to the broker.

Note
All this function needs to do is call __mqtt_recv and __mqtt_send every so often. I've picked 100 ms meaning that client ingress/egress traffic will be handled every 100 ms.
