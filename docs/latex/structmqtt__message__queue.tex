\subsection{Detailed Description}
A message queue. 

\begin{DoxyNote}{Note}
This struct is used internally to manage sending messages. 

The only members the user should use are {\ttfamily curr} and {\ttfamily curr\+\_\+sz}. 
\end{DoxyNote}
